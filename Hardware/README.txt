DistributedNetworkData/
    Gerber files for the board associated with the "Distributed Network Data" book

    EAGLE files for the PCB used in the DSL Workshop kit, associated with the 
    "Distributed Network Data" workshop by Alasdair Allan and Kipp Bradford.

StrataNY2012/
    Fritzing file for the boards used at Strata New York in 2012.
